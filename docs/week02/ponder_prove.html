Ponder and Prove

Tasks

Using Variables and Mathematical Operators

Define a function called \(exercise-math\). In this function, define the variables variables \(x, y\text{ and } z\). Then write the Clojure versions of the list of mathematical functions below and, using the println function you saw last week, print the value of each calculation.

\[\begin{align*} &x+y+z\\ &xyz\\ &x\div y\div z\\ &x^2+3y+z\\ &z^x+x^z\\ &\frac{z}{y}+x\\ &xyz+1\\ &xyz-1 \end{align*}\]

Using Logical Operators

Define a function called \(exercise-logic\). In this function, define the variables variables \(a, b\text{ and } c\) as numbers. Then write the Clojure versions of the list of mathematical comparisons below and, using the println function you saw last week, print the value of each comparison.

\[\begin{align*} &a>b\\ &b>a\\ &a < b\\ &b < c\\ &c\ge c\\ &a = c\\ &a\le b\\ &a < b \land b > c\text{ (in the context of logic, }\land\text{ means 'and'})\\ &a < b \lor b > c\text{ (in the context of logic, }\lor\text{ means 'or')} \end{align*}\]

Review

After completing the previous steps:

  1. Review one of the many possible solutions for this task,
  2. Compare and contrast your code with the possible solutions’ code, and
  3. Make note of any improvements you could make for future tasks.
